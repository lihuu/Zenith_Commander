# Zenith Commander 🚀

**Zenith Commander** 是一款专为 macOS 打造的原生双栏文件管理器。它将 Total Commander 的高效双栏设计与 Vim 的模态操作哲学完美融合，旨在为开发者和键盘极客提供极致的文件管理体验。
![alt text](image.png)

> **"极客的效率，原生的优雅。"**

## ✨ 核心特性

- **⚡️ 双面板架构 (Dual-Pane)**: 经典的左右分栏设计，文件搬运从未如此高效。
- **🎹 Vim 模态交互**: 告别鼠标，双手不离键盘。支持 Normal、Visual、Command 等多种模式。
- **📑 多标签页系统**: 支持多标签管理，工作区状态自动记忆。
- **🛠 极客工具箱**: 内置支持正则的批量重命名、快速终端启动、Git 状态集成（规划中）。
- **🎨 原生 SwiftUI**: 纯原生开发，轻量、流畅，完美适配 macOS Dark Mode。

## 📥 安装与构建

### 系统要求

- macOS 14.0 (Sonoma) 或更高版本
- Xcode 15.0+

### 本地构建

1. 克隆仓库:

   ```bash
   git clone https://github.com/lihuu/Zenith_Commander.git
   ```

2. 使用 Xcode 打开项目文件 `ZenithCommander.xcodeproj`。

3. 按下 `Cmd + R` 运行。

## 🎮 快捷键指南 (Cheat Sheet)

Zenith Commander 的核心在于键盘操作。以下是默认的键位映射：

### 🧭 导航 (Normal Mode)

| **按键**          | **功能**                     |
| ----------------- | ---------------------------- |
| `j` / `k`         | 上下移动光标                 |
| `l` / `Enter`     | 进入目录 / 打开文件          |
| `h` / `Backspace` | 返回上一级目录               |
| `Tab`             | 切换左右面板焦点             |
| `Shift + D`       | 切换驱动器/卷 (Drive Select) |

### 📝 编辑与操作

| **按键**                  | **功能**                             |
| ------------------------- | ------------------------------------ |
| `v`                       | 进入/退出 **Visual Mode** (多选模式) |
| `y`                       | 复制选中文件 (Yank)                  |
| `p`                       | 粘贴文件 (Paste)                     |
| `r`                       | **批量重命名** (仅在 Visual 模式下)  |
| `t`                       | 新建标签页                           |
| `w`                       | 关闭当前标签页                       |
| `Shift + L` / `Shift + H` | 切换下一个/上一个标签页              |

## 📸 功能预览

### 1. 批量重命名 (Batch Rename)

在 Visual 模式下选中文件并按 `r`，即可调出重命名面板。支持 `{n}` 计数器、`{date}` 日期变量以及正则表达式替换，并提供实时预览。

### 2. AI 文件分析

选中任意文件按下 `Shift + A`，Gemini 将根据文件名、类型及元数据，为你生成一份智能摘要，无需打开文件即可了解内容。
